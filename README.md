Sistema de Gestión de Inventario con Estructuras de Datos (Java)
Descripción general

Este proyecto es un Sistema de Gestión de Inventario basado en Java diseñado para simular operaciones reales de almacén utilizando múltiples estructuras de datos.
Demuestra la aplicación práctica de algoritmos y técnicas de organización de datos para gestionar productos, stock y transacciones de manera eficiente.

Conceptos Clave Aplicados
Programación Orientada a Objetos (POO)
Integración de Estructuras de Datos
Manejo de archivos para persistencia de datos
Diseño modular del sistema


Tecnologías Utilizadas
Java
Manejo de archivos (.txt)
Java Swing (JOptionPane)


Estructuras de Datos Utilizadas
Listas Enlazadas → Gestión dinámica de productos
Colas → Sistema de reposición de stock
Pilas → Seguimiento de ventas
Árboles Binarios → Búsqueda y ordenamiento eficiente
Matrices → Representación de la estructura del almacén


Funcionalidades
Gestión de Productos
. Agregar, actualizar y buscar productos
. Almacenamiento dinámico usando listas enlazadas
Control de Inventario
- Persistencia de datos utilizando archivos .txt
- Representación estructurada del almacén mediante matrices
Gestión de Stock
- Sistema de reposición implementado con colas
- Seguimiento eficiente de la disponibilidad de productos
Seguimiento de Ventas
- Identificación de los productos más vendidos usando pilas
- Mantiene un historial de movimientos de productos
Búsqueda Eficiente
- Uso de árboles binarios para una organización y búsqueda rápida de productos
